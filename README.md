# Expenses Next

![App Screenshot](https://i.imgur.com/yl4vG7R.png)

A modern expense tracker web app, inspired by [ExpenseOwl](https://github.com/tanq16/expenseowl), rebuilt with Next.js and HeroUI. This project features a refreshed UI, multi-user support, password-based authentication, multi-language support, and many additional improvements. It is fully compatible with data exported from ExpenseOwl.

## Features

- Built with Next.js for fast, modern web experience
- Beautiful UI using HeroUI and Tailwind CSS
- Multi-user support
- Password-based authentication
- Multi-language support (i18n)
- Compatible with ExpenseOwl data
- Enhanced dashboard, recurring expenses, reminders, and more

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/cmclient/expenses-next
   cd expenses-next
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Configuration
- See `config/` and `data/` folders for user and app configuration.
- Compatible with ExpenseOwl's exported data (see `data/` structure).
